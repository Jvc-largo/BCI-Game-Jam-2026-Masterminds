# Lucky Cipher

Lucky Cipher is a "Wordle"-like game built on Unity. It is designed for the BCI Game Jam 2026 as a
P300 EEG paradigm compatible video game. 

## Packages Used

- LSL4Unity Package :     https://github.com/labstreaminglayer/LSL4Unity.git
- BCI Essentials Package  https://github.com/kirtonBCIlab/bci-essentials-unity.git
